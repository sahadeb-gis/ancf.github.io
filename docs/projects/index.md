## Programmes

<div class="grid" markdown>

<div class="project-card" markdown>
![Community plantation](../assets/images/plantation%20infographic.png)

**[Community Plantation](community-plantation.md)**

A decade of community-led plantation across schools, colleges,
universities, religious institutions, and hill-tract villages — covering
more than 50 sites in 10+ districts.

`Native species` `Schools` `Universities` `Hill tracts`

[View Project →](community-plantation.md){ .md-button }
</div>

<div class="project-card" markdown>
![Tree tagging](../assets/images/placeholder-project.png)

**[Tree Tagging](tree-tagging.md)**

Labelling individual trees on campuses and in public spaces with their
scientific and local names — turning trees into teaching tools and the
starting point for inventories.

`Identification` `Campuses` `Education`

[View Project →](tree-tagging.md){ .md-button }
</div>

<div class="project-card" markdown>
![Mapping endangered plants](../assets/images/placeholder-project.png)

**[Mapping Endangered Plants](mapping-endangered-plants.md)**

A spatial inventory of endangered tree species across Bangladesh — the
foundation for any serious restoration work.

`GIS` `Remote sensing` `Endangered species`

[View Project →](mapping-endangered-plants.md){ .md-button }
</div>

<div class="project-card" markdown>
![Forest bathing](../assets/images/placeholder-project.png)

**[Forest Bathing](forest-bathing.md)**

Guided *shinrin-yoku* style nature-immersion programmes adapted to
Bangladeshi forests — wellbeing as a doorway to conservation.

`Wellbeing` `Nature experience` `Public engagement`

[View Project →](forest-bathing.md){ .md-button }
</div>

<div class="project-card" markdown>
![Science in conservation](../assets/images/placeholder-notebook.png)

**[Science in Conservation](science-in-conservation.md)**

Applying i-Tree, GIS, and remote sensing to quantify tree cover, ecosystem
services, and survival rates of planted seedlings.

`i-Tree` `GIS` `Python` `Monitoring`

[View Project →](science-in-conservation.md){ .md-button }
</div>

</div>
